    Проект marketplace(erc1155)
Проект представляет собой децентрализованный маркетплейс для торговли NFT стандарта ERC1155, развернутый на блокчейне Ethereum(тестовая сеть sepolia). Backend создан с использованием Hardhat для разработки, тестирования и развертывания смарт-контрактов. Frontend реализован на React (с использованием viem и wagmi) для обеспечения интерактивного пользовательского интерфейса.
Особенности платформы:
Поддержка торговли NFT стандарта ERC1155.
Интерактивный frontend с использованием React и библиотек для взаимодействия с блокчейном.

Состав проекта:
Backend (Hardhat): Содержит смарт-контракты, скрипты развертывания, тесты и конфигурацию Hardhat.
Frontend (React): Frontend приложение, включающее компоненты пользовательского интерфейса, интеграцию с кошельками (Metamask, WalletConnect), и взаимодействие со смарт-контрактами через viem и wagmi.
Общие файлы: Файлы конфигурации, зависимости и документация, общие для frontend и backend.

Технологии:
Смарт-контракты: Solidity, OpenZeppelin Contracts.
Backend: Hardhat.
Frontend: React, Vite, viem, wagmi, @mui/material,Styled Components.
Структура проекта:
Проект разделен на три основные директории: backend, frontend и общую корневую директорию. Каждая директория содержит свои собственные package.json и зависимости. В директории public (frontend) располагаются изображения и файлы JSON с метаданными для NFT.
